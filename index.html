NDA Reviewer Control Center

Operate the desktop deployment from the web

Launch the Python FastAPI backend (python app.py --web) on your desktop and use this interface to upload agreements, review the AI summary, approve the fill step, and download the completed package without leaving your browser.

⚠️ This tool provides an AI-assisted review; it is not legal advice. Always consult counsel for material agreements. No signing or submission occurs without explicit approval. Documents and profile data are handled locally; rotate keys and restrict access.

1. Connect to your running backend

The control center relays actions to the FastAPI server that runs on your desktop or another secure host. Update the API base URL if you are accessing the backend remotely.
