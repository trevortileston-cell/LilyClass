Heads up

Run the NDA Reviewer locally

This repository ships a Python-based NDA review workflow that is designed to run from the command line (with an optional FastAPI surface). The Vercel preview you are viewing is a static notice so that deployments do not show a blank screen.

Why nothing loads automatically

The reviewer coordinates file uploads, OpenAI Agents analysis, Google Sheets profile retrieval, PDF filling, and webhook calls. Those integrations require local credentials and are intentionally not executed on Vercel's serverless preview environment.

Try it yourself

  1. Clone this repo and create a virtual environment.
  2. Fill in .env (see .env.example for keys).
  3. Install dependencies with pip install -r requirements.txt.
  4. Run the CLI: python app.py --file ./examples/sample_nda.pdf and follow the prompts.

Prefer a browser? Launch the FastAPI variant locally with python app.py --web and point your browser to http://localhost:8000/docs.

Important disclaimers

  • This tool provides an AI-assisted review; it is not legal advice. Always consult counsel for material agreements.
  • No signing or submission occurs without explicit approval.
  • Documents and profile data are handled locally; rotate keys & restrict access.
Need to adapt this for hosted use? Wire a secure backend that can store and protect your OpenAI, Google, and webhook credentials before redeploying.
